# Moxt Skill 使用指南

## 获取更多 Skill

有三种方式扩展你的 AI 能力：

**1. 告诉 Momo 你的需求，让它帮你创建**
直接描述你想要的工作流，Momo 会生成一个专属 Skill：

```
Momo，帮我创建一个 Skill：每次我粘贴一段竞品描述，
自动输出竞品分析卡片，格式固定
```

**2. 找到开源 Skill，让 Momo 帮你安装**
在 [Clawhub](https://clawhub.ai/) 或 GitHub 上找到感兴趣的 Skill，把链接发给 Momo：

```
Momo，帮我安装这个 Skill：https://github.com/xxx/yyy
```

**3. 自己动手写**
在 `personal/System/Skills/` 或 `team/System/Skills/` 下新建文件夹，添加 `SKILL.md`，用自然语言描述执行逻辑即可。

***

## 预装 Skill 详细说明

### 🔬 deep-research-pro — 深度研究

**适合场景：** 需要调研一个话题，要求多来源、有引用、结论可信。

**工作方式：**

1. 把研究主题拆分为 3—5 个子问题
2. 每个子问题分别用不同关键词搜索，目标 15—30 个独立来源
3. 精读关键页面全文
4. 综合输出带引用的结构化报告

**示例提示词：**

```
/deep-research-pro 帮我深度研究"AI Agent 在企业软件中的落地现状"，
目标是做出投资决策参考
```

**说明：** 开始前会询问 1—2 个澄清问题（研究目的、深度偏好）；直接说"就按默认来"也可以跳过。

***

### ✍️ humanizer — 去 AI 味

**适合场景：** 把 AI 生成的文字润色得更自然，或检查自己的文章是否有 AI 腔。

**能识别并修复的 24 类 AI 写作模式（部分举例）：**

| 问题类型    | 典型 AI 写法                                | 修复后         |
| ------- | --------------------------------------- | ----------- |
| 过度强调重要性 | "这标志着一个关键转折点"                           | 删除或改为具体事实   |
| AI 词汇堆砌 | delve / pivotal / landscape / testament | 换成日常表达      |
| 破折号滥用   | 句子里大量 —                                 | 改用逗号或重组句子   |
| 负向排比    | "不只是…更是…"                               | 直接说结论       |
| 三连排比    | "高效、智能、现代"                              | 选一个最重要的     |
| 服务性语气   | "希望对你有帮助！"                              | 直接删除        |
| 无观点写作   | 中立列举无判断                                 | 加入真实立场和具体细节 |

**示例提示词：**

```
/humanizer 帮我处理下面这段文字，让它听起来不像 AI 写的：
[粘贴文字]
```

***

### 📈 marketing-mode — 营销策略模式

**适合场景：** 需要制定营销方案、写文案、做 SEO、提升转化率、规划付费广告时激活。

**激活后 AI 会扮演营销策略师角色，调用以下知识：**

* **140+ 营销打法**：SEO、内容、社媒、邮件、付费广告、联盟营销、产品增长……

* **70+ 心理学模型**：锚定效应、损失厌恶、社会证明、稀缺感、IKEA 效应……

* **5 阶段产品上线框架**：内测 → Alpha → Beta → Early Access → 正式发布

* **定价策略**：竞品分析、价值定价、分层套餐、付费转化优化

* **CRO 转化优化**：落地页、注册流程、邮件序列、A/B 测试

**示例提示词：**

```
/marketing-mode 我有一个 B2B SaaS 产品，月付 $49，目前主要靠口碑，
想系统做增长，预算有限，帮我出一个 90 天营销计划
```

***

### 📰 news-summary — 新闻摘要

**适合场景：** 快速了解今日全球重要新闻，支持文字摘要。

**数据来源：**

* BBC（世界、财经、科技）

* 路透社

* NPR（美国视角）

* 半岛电视台（全球南方视角）

**输出格式：**

```
📰 今日新闻 [日期]

🌍 国际
- [要点1]
- [要点2]

💼 财经
- [要点1]

💻 科技
- [要点1]
```

**示例提示词：**

```
/news-summary 给我今天的新闻摘要，重点关注科技和商业
```

***

### 🎬 openclaw-youtube-transcript — YouTube 字幕提取

**适合场景：** 不想看完整视频，直接拿逐字稿；需要对视频内容做总结、分析、引用。

**支持：** 手动字幕 + YouTube 自动生成字幕，可指定语言。

**示例提示词：**

```
/openclaw-youtube-transcript https://www.youtube.com/watch?v=xxxxx
帮我提取这个视频的字幕，然后总结核心观点
```

**语言指定：**

```
提取这个视频的中文字幕：https://...
```

***

### 🧾 summarize — 内容总结

**适合场景：** 快速消化一个网页、PDF 文件、音频或 YouTube 链接，不想逐字阅读时使用。

**支持的输入类型：**

* 网页 URL

* 本地 PDF 文件

* 音频文件

* YouTube 链接

**示例提示词：**

```
/summarize https://example.com/long-article 用中文给我一个 300 字的摘要
```

```
/summarize 这份 PDF 报告的核心结论是什么？@市场调研报告.pdf
```

**长度控制：** 可指定 short / medium / long，或直接说字数要求。

***

### 💹 yahoo-finance — 股票与财经数据

**适合场景：** 查股价、看财务指标、对比多支股票、了解分析师观点，无需任何 API Key。

**支持的查询类型：**

| 命令    | 说明             | 示例                   |
| ----- | -------------- | -------------------- |
| 股价    | 实时报价           | `查一下 AAPL 的股价`       |
| 详细报价  | 涨跌、成交量等        | `MSFT 今天怎么样`         |
| 财务基本面 | PE、EPS、利润率、ROE | `NVDA 的估值指标`         |
| 财报    | 下次财报日期、历史 EPS  | `TSLA 下次财报什么时候`      |
| 股息    | 股息率、除息日        | `KO 的股息信息`           |
| 分析师评级 | 买/持/卖分布、近期调整   | `AAPL 分析师怎么看`        |
| 期权链   | 看涨/看跌期权、行权价    | `SPY 的期权数据`          |
| 历史走势  | 1天～10年         | `GOOGL 过去1年的走势`      |
| 多股对比  | 并排比较关键指标       | `对比 AAPL MSFT GOOGL` |

**支持的市场：** 美股、印度 NSE/BSE、加密货币（BTC-USD）、外汇、ETF

**示例提示词：**

```
/yahoo-finance 帮我对比英伟达、AMD、高通三家公司的基本面
```

```
/yahoo-finance 查一下 BTC-USD 过去30天的走势
```

## 使用建议

* **不确定用哪个 Skill？** 直接描述你的需求，AI 会自动判断是否需要调用某个 Skill。

* **组合使用效果更好：** 例如用 `openclaw-youtube-transcript` 提取视频字幕，再用 `humanizer` 润色整理成文章。

* **自定义 Skill：** 如果以上 Skill 无法满足你的需求，可以在 `team/System/Skills/` 下创建自己的 Skill，详见平台帮助文档。
